Computational Photography (CS691B)
Project #1: Hybrid Images [Code]

Hybrid images are static images that changes interpretation as a function of the viewing distance. In general, for human vision, only the low-frequency content (e.g., texture) part of the image can be seen at a distance, but, high-frequency components (e.g., edges) dominant our interpretation when looking at a closer distance. This project is to created such image illusion.

Project #2: Seam Carving for Content-Aware Image Resizing [Code]

Seam carving is a technique used for image resizing. Unlike traditional method (i.e., cropping, stretching), seam caving add or remove insignificant seams in a image. A seam in a image is defined as a pixel path from top to bottom or left to right. Moreover, Each seam only occupies one pixel in each row or column, and the successor of each pixel has to be it 8-connected neighbor. The selection and ordering of removing or adding seams are calcuated by a graph energy function. Dynamic programming is also used to speed up seam searching process.
