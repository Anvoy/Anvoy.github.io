Introduction

Hybrid images are static images that have different interpretations and usually changes as a function of the viewing distance. This illusion is created by fuse two images with different spectrum together. In general, the low frequency spectrum image is generated by passing image through a low-pass filter, and the high frequency spectrum image is generated by passing image through a high-pass filter. The final hybrid image is created by summing the outputs of two filters. In this assignment, hybrid image generating algorithm has been implemented and several example results are shown next.

Reference

Oliva, Aude, Antonio Torralba, and Philippe G. Schyns. "Hybrid images." ACM Transactions on Graphics (TOG). Vol. 25. No. 3. ACM, 2006.

As we can observe, this example does not illustrate two interpretations. There are a few reasons:

  • Bear image only contain a small amount of high frequency information. The filtered bear image does not contain significant amount of information to create the illusion.
  • Alignment issue. In the bear image, bear has a longer face which exceed the face region of the cat. As the result you only see the nose of the bear in the hybrid image.

Color VS Gray

To compare the difference of color and gray images, the steve-gates example is used here. Four hybrid images are presented to visualize the difference.

Hybrid image of Steve Jobs and Bill Gates when both filtered images are in gray scale

Hybrid image of Steve Jobs and Bill Gates when low frequency spectrum is a gray image (Bill Gates) and High frequency Spectrum is a color image(Steve Jobs)

Hybrid image of Steve Jobs and Bill Gates when low frequency spectrum is a color image (Bill Gates) and High frequency Spectrum is a gray image(Steve Jobs)

Hybrid image of Steve Jobs and Bill Gates when both filtered images are color images

As we can observe, high frequency spectrum color image does not affect the perception of hybrid image significantly. on the contrary, low frequency spectrum color image enhance the details of the hybrid image therefore produces a better hybrid image.
